# dotfiles for my linux programs

## I use [chezmoi](https://www.chezmoi.io/) to manage my dotfiles

- OS : **Arch Linux**
- Shell: **zsh**
- WM : **bspwm**
- Bar : **polybar**
- App luancher: **rofi**
- notification: **dunst**
- GTK theme: **Dracula**
- Editor: **Neovim and VS Codium**

#### All my programs use the [catppuccin](https://github.com/catppuccin/catppuccin) theme.
